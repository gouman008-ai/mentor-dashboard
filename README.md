# 🎓 Smart Student Performance System – Mentor Dashboard

## 📌 Overview
This project is a simple **Mentor Dashboard** that helps faculty track student performance across three assessments:
- CAT1
- CAT2
- FAT

It allows quick insights into student performance including totals, averages, and grades.

---

## 🚀 Features

### ✅ Student Management
- Add new student
- Validate marks (0–100 range)
- Display all student records

### 📊 Performance Calculation
- Total marks
- Average marks
- Grade calculation:
  - A (≥ 90)
  - B (≥ 75)
  - C (≥ 50)
  - Fail (< 50)
